01. 氣體力學原理

[ Dynamic Simulation: Active ]

風扇透過葉片旋轉將電能轉化為動能。依據流體力學，葉片傾角會導致背風面壓力低於迎風面，產生氣流推動力。

// Bernoulli's Simplified Flow

$$ P = \frac{1}{2} \rho v^2 $$

02. 馬達構造：內轉子 vs 外轉子

直流無刷馬達 (BLDC) 根據磁鐵旋轉的位置，主要分為內轉子與外轉子兩種設計。散熱風扇為了獲得更大的轉矩與安裝葉片的便利性，絕大多數採用外轉子 (Outer Rotor) 設計。

內轉子 (Inner Rotor)

  • • 轉子位於定子內側，轉動慣量小。
  • • 起動速度快，動態響應優異。
  • • 適用於精密控制（如伺服電機）。

外轉子 (Outer Rotor)

  • • 轉子位於外部，磁鐵體積可較大。
  • • 旋轉平穩且能提供較大轉矩。
  • • 製程穩定，散熱風扇主流。

03. PWM 控制與轉速計算

在實際量測中，風扇轉動一圈會產生兩個波形的脈衝。當 Duty Cycle 越大，馬達轉矩增加，波形變得更加密集，完成一圈的總時間 $T_1+T_2+T_3+T_4$ 將縮短。

低轉速 (Duty 20%) 週期較長
高轉速 (Duty 80%) 週期極短

// RPM Calculation

$$ RPM = \frac{60}{T_1 + T_2 + T_3 + T_4} $$
